7 `Device` is a value-type replacement of [`UIDevice`](https://developer.apple.com/library/ios/documentation/UIKit/Reference/UIDevice_Class/).
8 
9 ##Features
10 
11 - [x] Device identification
12 - [x] Device family detection
13 - [x] Device group detection
14 - [x] Simulator detection
15 - [x] Battery state
16 - [x] Battery level
17 - [x] Equatable
18

24 ## Installation
25 DeviceKit can be installed in various ways.
26 
27 ### CocoaPods
28 
29 #### Swift 3.0
30 ```ruby
31 pod 'DeviceKit', '~> 1.0'
32 ```
33 #### Swift 2.3 (Unsupported)
34 ```ruby
35 pod 'DeviceKit', :git => 'https://github.com/dennisweissmann/DeviceKit.git', :branch => 'swift-2.3-unsupported'
36 ```
37

49 ### Manually
50 To install it manually drag the DeviceKit project into your app project in Xcode or add it as a git submodule.
51 In your project folder enter:
52 ```bash
53 $ git submodule add https://github.com/dennisweissmann/DeviceKit.git
54 ```

56 ## Usage
57 First make sure to import the framework:
58 ```swift
59 import DeviceKit
60 ```
61 
62 Here are some usage examples. All devices are also available as simulators:
63 ```swift
64 .iPhone6 => .simulator(.iPhone6)
65 .iPhone6s => .simulator(.iPhone6s)
66 ```
67 
68 ### Get the Device You're Running On
69 ```swift
70 let device = Device()
71 
72 print(device) // prints, for example, "iPhone 6 Plus"
73 
74 if device == .iPhone6Plus {
75  // Do something
76 } else {
77  // Do something else
78 }
79 ```
80 
81 ### Get the Device Family
82 ```swift
83 let device = Device()
84 if device.isPod {
85  // iPods (real or simulator)
86 } else if device.isPhone {
87  // iPhone (real or simulator)
88 } else if device.isPad {
89  // iPad (real or simulator)
90 }
91 ```
92 
93 ### Check If Running on Simulator
94 ```swift
95 let device = Device()
96 if device.isSimulator {
97  // Running on one of the simulators(iPod/iPhone/iPad)
98  // Skip doing something irrelevant for Simulator
99 }
100 ```
101 
102 ### Get the Simulator Device
103 ```swift
104 let device = Device()
105 switch device {
106 case .simulator(.iPhone6s): break // You're running on the iPhone 6s simulator
107 case .simulator(.iPadAir2): break // You're running on the iPad Air 2 simulator
108 default: break
109 }
110 ```
